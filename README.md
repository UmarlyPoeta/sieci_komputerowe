# Sieci Komputerowe

Repozytorium zawiera materiały z laboratoriów oraz kolokwiów z przedmiotu **Sieci Komputerowe**.

## Struktura repozytorium

### Laboratoria

| Katalog | Opis |
|---------|------|
| `lab02` | Interfejsy sieciowe – analiza konfiguracji sieci (`ip addr`, wyniki) |
| `lab03` | Skanowanie sieci – nmap, p0f, routing, resolv |
| `lab04` | Tablice routingu |
| `lab05` | Porównanie szyfrowania i kompresji (skrypt Python + wykres) |

### Kolokwia

| Katalog | Opis |
|---------|------|
| `kolokwium1` | Kolokwium 1 – WiFi, Ethernet, adres MAC, iperf |
| `kolokwium2` | Kolokwium 2 – nmap, NAT, Wireshark, sockety |
| `kolokwium3` | Kolokwium 3 – interferencja, dudnienie, VPN, SSH, OSPF, RIP |
